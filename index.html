Windows Troubleshooting Cheatsheet

A collection of built-in tools and commands for diagnosing and fixing Windows issues.

Networking

ipconfig /all

Displays full TCP/IP configuration for all adapters, including MAC address and DHCP info.

ping 8.8.8.8

Tests connectivity to a specific IP address or domain to check for packet loss.

tracert google.com

Traces the route packets take to a destination, identifying network hops and delays.

netstat -ano

Shows active connections, listening ports, and the owning Process ID (PID).

ipconfig /flushdns

Clears the local DNS resolver cache. Use this if websites aren't resolving correctly.

ipconfig /renew

Releases and renews the IP address from the DHCP server. Good for connection issues.

ncpa.cpl

Opens the Network Connections panel to manage network adapters.

System File & Health

sfc /scannow

System File Checker. Scans and repairs corrupted or missing protected Windows system files.

DISM /Online /Cleanup-Image /RestoreHealth

Repairs the underlying Windows component store, which `sfc` relies on. Run this if `sfc` fails.

chkdsk /f /r C:

Checks a disk for file system errors (/f) and locates bad sectors to recover data (/r).

resmon

Opens the Resource Monitor, providing a detailed real-time view of CPU, memory, disk, and network usage.

shutdown /r /o /t 0

Restarts the PC into the Advanced Startup Options menu for Safe Mode, System Restore, etc.

mrt

Opens the Malicious Software Removal Tool to scan for and remove specific, prevalent malware.

verifier

Opens Driver Verifier Manager to stress-test drivers for instability. (Advanced users only).

perfmon /rel

Opens the Reliability Monitor to view a history of system stability and application crashes.

System Information

systeminfo

Displays a detailed configuration summary of the computer's hardware and software.

msinfo32

Opens the System Information GUI, a comprehensive tool for viewing hardware, components, and software.

dxdiag

DirectX Diagnostic Tool. Gathers information about DirectX components and drivers for multimedia troubleshooting.

driverquery

Displays a list of all installed device drivers and their properties.

sysdm.cpl

Opens System Properties to manage computer name, performance settings, and restore points.

gpresult /r

Displays the Resultant Set of Policy (RSoP) for the logged-in user and computer.

Process & Service Management

taskmgr

Opens the Task Manager to view running processes, performance, and startup apps.

tasklist

Displays a list of currently running processes on the local or a remote machine.

taskkill /F /PID 1234

Forcefully terminates a process by its Process ID (PID). Use `tasklist` to find the PID.

services.msc

Opens the Services management console to start, stop, and configure Windows services.

taskschd.msc

Opens the Task Scheduler to view and manage automated tasks.

Get-Process | Sort CPU -Desc | Select -First 10

PowerShell: Lists the top 10 processes by CPU usage.

Stop-Process -Name "notepad"

PowerShell: Stops a process by its name (e.g., "notepad").

Get-Service

PowerShell: Lists all services on the system and their status.

Start-Service "Spooler"

PowerShell: Starts a service by name (e.g., the "Spooler" print service).

Disk & Storage Management

diskmgmt.msc

Opens the Disk Management GUI to manage partitions, format drives, and assign drive letters.

diskpart

Opens the command-line disk partitioning utility. (Advanced users only).

cleanmgr

Opens the Disk Cleanup utility to free up space by removing temporary files.

dfrgui

Opens the Drive Optimizer to defragment HDDs or optimize (TRIM) SSDs.

robocopy

Robust File Copy. A powerful tool for copying files and directories. (e.g., `robocopy C:\src D:\dest /E`)

Get-PhysicalDisk | ? {$_.HealthStatus -ne 'Healthy'}

PowerShell: Checks all physical disks and lists any that are not 'Healthy'.

Useful Utilities & Applets

eventvwr.msc

Opens the Event Viewer, which shows a log of system and application errors and events.

compmgmt.msc

Opens Computer Management, a console with many tools (Event Viewer, Disks, Services, etc.).

devmgmt.msc

Opens the Device Manager to view and manage hardware and drivers.

msconfig

Opens System Configuration to manage boot options, services, and startup programs.

control

Opens the classic Control Panel.

appwiz.cpl

Opens "Programs and Features" to quickly uninstall applications.

perfmon

Opens the Performance Monitor to analyze system performance and resource usage in detail.

mstsc

Opens the Remote Desktop Connection client.

inetcpl.cpl

Opens Internet Properties to manage cache, security, and proxy settings.

powercfg.cpl

Opens Power Options to manage power plans and sleep settings.

Security & Administrative Tools

gpedit.msc

Opens the Local Group Policy Editor (Not in Windows Home edition).

regedit

Opens the Registry Editor. (Use with caution).

secpol.msc

Opens the Local Security Policy editor (Not in Windows Home edition).

netplwiz

Opens the advanced User Accounts control panel (also `control userpasswords2`).

lusrmgr.msc

Opens Local Users and Groups manager (Not in Windows Home edition).

certmgr.msc

Opens the Certificate Manager for the current user.

printmanagement.msc

Opens the Print Management console to manage printers and print servers.

gpupdate /force

Forces an immediate update and reapplication of all Group Policy settings.

PowerShell Specific

Get-Help Get-Process

PowerShell: Shows detailed help for any cmdlet (command).

Update-Help

PowerShell: Downloads the latest help files for PowerShell cmdlets. (Run as Admin).

Get-Command

PowerShell: Lists all available commands, cmdlets, and functions.

Measure-Command { Get-Process }

PowerShell: Measures how long a command or script block takes to execute.

Accessibility & Misc. Tools

snippingtool

Opens the Snipping Tool (or Snip & Sketch) for taking screenshots.

charmap

Opens the Character Map to find and copy special characters.

magnify

Toggles the Magnifier accessibility tool.

osk

Opens the On-Screen Keyboard.

sndvol

Opens the Volume Mixer to control audio levels for different applications.

timedate.cpl

Opens the Date and Time settings panel.

Use these commands in Command Prompt or PowerShell, preferably as an Administrator.

Commands marked PowerShell must be run in a PowerShell window. Commands ending in .msc or .cpl open GUI windows.
